# Contact Book
Contact Book is an simple application for managing contacts.

## Running
- API
  - Create database
  - Configure database in .env file
  - Execute command ```php artisan migrate```
  - Then ```php artisan serv``` to start API
- Web frontend
  - Execute command ```npm start``` or ```yarn start```

## Stack
- Laravel
- React
- TypeScript
- Axios
- React-router-dom
- Material-ui
